PanelApp / HPO / GO Tools — Shiny (R) App

This repository contains an R Shiny application for: (1) PanelApp gene enrichment, (2) discovery panels from HPO / gene lists / GO keywords, and (3) a gene-based network explorer (HPO / GO / PanelApp phenotypes).

Note: Shiny apps are not static HTML pages. GitHub Pages can host this landing page, but the interactive app must be run via R (locally) or hosted on a Shiny service (e.g., shinyapps.io, Shiny Server, Posit Connect, or a Docker deployment).

Run locally

Clone the repo and run the app from R/RStudio:

git clone <YOUR_GITHUB_REPO_URL>
cd <REPO_FOLDER>

# In R:
install.packages(c("shiny","shinyjs","shinybusy","tidyverse","DT","visNetwork",
                   "httr","jsonlite","readxl","writexl","pheatmap","matrixStats",
                   "ggVennDiagram","conflicted","fgsea","msigdbr"))
shiny::runApp()
View app.R Project README (HTML)

Host the app (recommended)

To make the app accessible from a browser, deploy it to a Shiny host.

shinyapps.io
Quickest hosted option for personal projects.
Docker
Good for reproducible deployments and servers.
Shiny Server
Self-hosted, runs on a Linux server/VM.

Once hosted, you can link it here (and optionally embed it with an iframe).

How to use on GitHub

You can upload app.R + this index.html to your repo. If you enable GitHub Pages, it will serve this landing page. The interactive Shiny app still needs hosting (see above).

Tip: After you deploy the app (e.g., shinyapps.io), paste the URL into the iframe section above.
